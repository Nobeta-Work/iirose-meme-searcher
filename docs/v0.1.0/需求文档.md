# iirose-meme-searcher 蔷薇表情包搜索器

## 核心需求

**核心功能**：
- 模仿qq表情包功能，当用户输入字符时，可以调用表情包搜索器，显示出可选的表情包图片列表，点击即可发送。
- 插件，允许用户js葱终端导入，例如https://shadowsee.icu/upload/iirose-scdn.js

**前端需求**：
- 要求iirose适配前端面板
- 支持配置，包括前缀关键词
